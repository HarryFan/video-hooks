# Video Hooks · 短影音開場 Hook 靈感 Skill

> Hook 的本質：讓觀眾在前 1–3 秒決定「繼續看」而不是「滑走」。
> Transitional Hook = 剪輯過渡的那個瞬間，用「意外感」或「動態衝擊」釘住注意力。

## Hook 概念說明

**Transitional Hook** = 在影片最開頭（前 1–3 秒）使用的過渡短片素材，透過：
- 🎯 **意外感**（跌倒、爆炸、奇怪行為）
- 🐾 **情感觸發**（可愛動物、搞笑反應）
- ⚡ **動態節奏**（快速切換、衝擊畫面）

讓觀眾大腦無法「滑走」，自動停下來看。

**核心法則：**
1. Hook 本身要跟你的內容「主題相符」或「情緒相符」
2. Hook 結束後，你的第一句話要接住它（別讓 hook 死在那裡）
3. 越奇怪越好，越讓人問「這是什麼？！」越有效

## 中文 Hook 創作公式庫

> 以下是文字腳本公式，搭配上方視覺素材使用。
> 核心原則：前 3 秒說完，10–14 個字內，疊加 2 個以上心理觸發器效果最強。

### 觸發機制說明
| 觸發器 | 作用 | 對應情緒 |
|--------|------|---------|
| 好奇心 | 製造資訊缺口，觀眾想知道答案 | 「然後呢？」 |
| 打斷慣性 | 顛覆預期，大腦無法略過 | 「等等，什麼？」 |
| 自我相關 | 觀眾覺得「這說的是我」 | 「這我懂」 |
| 情緒喚起 | 觸發恐懼、興奮、共鳴、憤怒 | 「我有感覺」 |

---

### 一、好奇心型（製造資訊缺口）

**1. 反常識宣言**
```
模板：「你以為的[X]，其實是錯的。」
例：「你以為每天發文就能漲粉，其實是錯的。」
例：「你以為努力就夠了，其實是錯的。」
觸發：好奇心 + 打斷慣性
```

**2. 差點發生的事**
```
模板：「我差點[糟糕結果]，直到我發現了這件事⋯⋯」
例：「我差點放棄自媒體，直到我改了一個習慣。」
例：「我差點被公司開除，直到我學會這一招。」
觸發：好奇心 + 情緒喚起
```

**3. 業內秘密**
```
模板：「[專家/業內人] 不想讓你知道的[X]。」
例：「品牌方不想讓你知道的合作報價潛規則。」
例：「演算法不想讓你知道的發文時間秘密。」
觸發：好奇心 + 自我相關
```

**4. 倒數預告**
```
模板：「第[N]點，我用了之後人生完全不一樣。」
例：「五個省錢技巧，第三點我用了之後每個月多存一萬。」
觸發：好奇心（吊胃口）
```

**5. 不可能的結果**
```
模板：「我在[短時間]內[極端成果]，靠的是這個。」
例：「我三個月從零漲到一萬粉，全靠這一個改變。」
例：「我花三千塊，賺回了三十萬，方法在這裡。」
觸發：好奇心 + 情緒喚起
```

**6. 「如果我告訴你⋯⋯」**
```
模板：「如果我告訴你[反直覺事實]，你信嗎？」
例：「如果我告訴你，少發文反而漲更快，你信嗎？」
觸發：好奇心 + 打斷慣性
```
