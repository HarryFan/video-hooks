# Video Hooks · 短影音開場 Hook 靈感 Skill

> Hook 的本質：讓觀眾在前 1–3 秒決定「繼續看」而不是「滑走」。
> Transitional Hook = 剪輯過渡的那個瞬間，用「意外感」或「動態衝擊」釘住注意力。

## Hook 概念說明

**Transitional Hook** = 在影片最開頭（前 1–3 秒）使用的過渡短片素材，透過：
- 🎯 **意外感**（跌倒、爆炸、奇怪行為）
- 🐾 **情感觸發**（可愛動物、搞笑反應）
- ⚡ **動態節奏**（快速切換、衝擊畫面）

讓觀眾大腦無法「滑走」，自動停下來看。

**核心法則：**
1. Hook 本身要跟你的內容「主題相符」或「情緒相符」
2. Hook 結束後，你的第一句話要接住它（別讓 hook 死在那裡）
3. 越奇怪越好，越讓人問「這是什麼？！」越有效

## Hook 接法公式（如何用 Hook 開頭）

Hook 只是「門」，門後的第一句話決定留存率。

### 公式 A：意外反差
```
[播 Hook：Treadmill Fall]
開口說：「一年前，我的事業就是這樣倒的——但我後來做到了XXX...」
```

### 公式 B：直接提問
```
[播 Hook：No Trust]
開口說：「你相信嗎？我用這個方法，三個月漲了XX粉...」
```

### 公式 C：情境帶入
```
[播 Hook：Dog Crash]
開口說：「我家的狗告訴了我一個關於堅持的道理——聽起來蠢，但真的有用。」
```

### 公式 D：懸念製造
```
[播 Hook：Plane Crash]
開口說：「這是我生命中最崩潰的一天。以下是完整故事。」
```
